# Laravel Movie App

Aplikasi web untuk menampilkan informasi film menggunakan **Laravel 5.5**, **OMDb API**, dan **Tailwind CSS**.

Sistem ini memungkinkan pengguna untuk mencari film, melihat detail film, menyimpan film favorit, dan menghapusnya.
